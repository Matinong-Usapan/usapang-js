### Project Setup Log

This is the documentation for the series of steps taken to setup the main structure of the monorepo. To be used as reference for possible future updates or for troubleshooting potential issues on the project structure, not as a standard of correctness.

1. Install Node (v24.11.0) including npm (v11.6.1) using the installer from the Node website
2. Go to / Create project repository and run <code>npm init</code> and provide repo details. (Make sure the package type is "module")
3. Manually update the monorepo **project.json** to remove the scripts (for now*), unspecify the main file, and configure the workspaces:

```json
{
  "name": "@usapang-js/root",
  "workspaces": [
    "./backend-app/",
    "./frontend-app/",
    "./common/lib/*"
  ]
}
```
4. Create a **package.json** file in each of the workspaces following this format:
  ```json
  {
    "name": "@usapang-js/<URL format workspace name>",
    "version": "1.0.0",
    "description": "<workspace description>",
    "type": "module"
  }
  ```
5. Run <code>npm create vite@latest frontend-app</code>, and choose **React** > **Typescript** > **No** > **No**
6. Run <code>cd .\\frontend-app\\</code>, <code>npm i</code>
7. Migrate generic configs from the **frontend-app** package (generated by Vite) into the monorepo root:
- Migrated these dependencies from and to the corresponding **package.json** files
  ```json
  "devDependencies": {
      "@eslint/js": "^9.36.0",
      "eslint": "^9.36.0",
      "globals": "^16.4.0",
      "typescript": "~5.9.3",
      "typescript-eslint": "^8.45.0"
    }
  ```
- Modify **frontend-app** > **eslint.config.js** to look like this:
  ```
  import baseConfig from '../eslint.config.js'
  import reactHooks from 'eslint-plugin-react-hooks'
  import reactRefresh from 'eslint-plugin-react-refresh'

  export default [
    ...baseConfig,
    {
      files: ['**/*.{ts,tsx,jsx}'],
      plugins: {
        'react-hooks': reactHooks,
        'react-refresh': reactRefresh,
      },
      rules: {
        ...reactHooks.configs['recommended-latest'].rules,
        ...reactRefresh.configs.vite.rules,
      },
    },
  ]
  ```
- Create **eslint.config.js** in root folder:
  ```
  import js from '@eslint/js'
  import globals from 'globals'
  import tseslint from 'typescript-eslint'
  import { defineConfig, globalIgnores } from 'eslint/config'

  export default defineConfig([
    globalIgnores(['dist', 'dist-ssr', 'node_modules']),
    {
      files: ['**/*.{js,ts,jsx,tsx}'],
      extends: [
        js.configs.recommended,
        tseslint.configs.recommended,
      ],
      languageOptions: {
        ecmaVersion: 2020,
        globals: globals.browser,
      },
    },
  ])
  ```
- Move the **.eslintignore** file from the **frontend-app** workspace into the root directory
8. Run <code>npm i</code> in the monorepo root directory
